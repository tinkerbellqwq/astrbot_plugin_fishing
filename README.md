# AstrBot 钓鱼系统插件

一个功能齐全的钓鱼系统插件，为AstrBot添加丰富多彩的钓鱼游戏体验。

## 功能特点

- **钓鱼系统**：基础钓鱼、自动钓鱼、使用鱼饵
- **物品系统**：鱼竿、鱼饵、装饰品等多种装备
- **商店系统**：购买鱼饵、鱼竿等物品
- **抽卡系统**：单抽和十连抽
- **成就系统**：完成各种钓鱼任务获得奖励
- **每日签到**：每日签到获得奖励
- **背包管理**：鱼类收集、物品管理

## 指令列表

### 基础命令
- **注册**：注册钓鱼用户
- **钓鱼/fish**：进行一次钓鱼
- **签到/signin**：每日签到领取奖励
- **金币/coins**：查看当前金币

### 背包相关
- **背包/bag/inventory**：查看鱼类背包
- **鱼饵/baits**：查看鱼饵背包
- **鱼竿/rods**：查看鱼竿背包

### 商店与购买
- **商店/shop**：查看可购买的物品
- **购买鱼饵 ID/buybait ID**：购买指定ID的鱼饵
- **购买鱼竿 ID/buyrod ID**：购买指定ID的鱼竿
- **使用鱼饵 ID/usebait ID**：使用指定ID的鱼饵

### 出售鱼类
- **出售/sell**：出售背包中所有鱼
- **出售稀有度 <1-5>/sellr <1-5>**：出售特定稀有度的鱼

### 抽卡系统
- **抽卡 ID/gacha ID**：进行单次抽卡
- **十连 ID/multi ID**：进行十连抽卡

### 其他功能
- **自动钓鱼/auto**：开启/关闭自动钓鱼功能
- **排行榜/rank**：查看钓鱼排行榜
- **帮助/help**：显示钓鱼游戏帮助信息

## 安装说明

1. 将插件文件夹放入AstrBot的`data/plugins/`目录下
2. 重启AstrBot或使用插件加载命令加载插件
3. 在聊天中使用`注册`命令开始游戏

## 游戏说明

1. **钓鱼系统**：
   - 钓鱼会随机获得不同稀有度的鱼
   - 鱼的稀有度从1到5星不等，星级越高价值越高
   - 使用更好的鱼竿和鱼饵可以提高获得高稀有度鱼的几率

2. **装备系统**：
   - 鱼竿：提高钓鱼成功率和获得高稀有度鱼的几率
   - 鱼饵：提供特殊加成效果，有使用时限
   - 饰品：提供各种属性加成

3. **抽卡系统**：
   - 通过金币或钻石在不同的卡池中抽取稀有物品
   - 有单抽和十连抽两种方式
   - 十连抽有更高几率获得稀有物品

4. **成就系统**：
   - 完成各种钓鱼任务获得奖励
   - 奖励包括金币、称号、鱼饵等
